# DaxBot
My Discord bot. Current working version is not open source atm, this is the 2026 rewrite. 
Coded w/ discord.js

## Requirements
I mean... this is more just for transparency. If you wanna run it yourself, be my guest, just don't claim to be me...
- Node.js LTS (Latest) 
    - At time of writing, the latest LTS version is v24.14.0
- FFMPEG
    - Available in PATH
 
For what it's worth I'm running this on Windows atm but I'm trying to have it be less dependent on that so I can have flexible hosting options.
PM2 is also helpful.
